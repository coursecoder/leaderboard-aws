About Us

Welcome! This project deploys a fully serverless game leaderboard using AWS resources: S3, API Gateway, Lambda, IAM, and DynamoDB. The result is a highly available, scalable, and secure infrastructure with low pricing, fast response times, and reliable uptime.

Documentation

Spin up your own leaderboard using the full setup guide on GitHub. You'll be up and running in minutes.
01

Clone the Repo

git clone https://github.com/coursecoder/leaderboard-aws.git
02

Install Dependencies

pip install -r requirements.txt
03

Run Setup Script

chmod +x ./resources/setup.sh && ./resources/setup.sh
04

Tear Down

chmod +x ./resources/teardown.sh && ./resources/teardown.sh
